Собственные плагины, код написан в одну строку без лишних проверок и логов. Код можно через ИИ привести в читаемый вид

1.js — это контейнер плагинов. После установки в настройках приложения Lampa появляется раздел "Плагины", где можно включать или отключать каждый плагин индивидуально

Состав контейнера

Online Mod (от nb557) — лучший источник онлайн-просмотра фильмов и сериалов 

Card Rating — Плагин выводит на карточку фильма рейтинг с КиноПоиска с логотипом. Если рейтинг не найден, отображается стандартный рейтинг с TMDB. Также добавляет в карточку рейтинг КиноПоиска. Плагин пытается минимизировать API запросы к кинопоиску, рекомендуется использовать свой ключ с kinopoiskapiunofficial.tech

Content Labels — Добавляет на постер метки о жанре контента (аниме,мультфильм,реалити,драма): синяя метка — фильм, красная — сериал

Lampa Tweaks — Различные улучшения и отключения: убирает рекламу, лишние кнопки и уведомления, добавляет цветные метки на торрентах (раздают/качают) для лучшей читаемости

Russian Content — Создаёт категорию "Русское" с линиями контента: Сейчас смотрят (популярное за 1 месяц), Популярное за год, Фильмы, Сериалы, Мультфильмы, Реалити-шоу, Документальные....

Torrent Search — Добавляет кнопку в шапке при входе в "Торренты" в карточке фильма (позволяет быстро перевыбрать Jackett при ошибке или не загрузке страницы), а также кнопку "Парсер" в меню Действие (долгий тап на карточке), удобно для DMCA контента

Torrserver Switch — Подставляет из списка общедоступных серверов бесплатный TorrServer для просмотра торрентов. Для использования необходимо: включить встроенный клиент, включить дополнительную ссылку. Сервер автоматически переподключается при входе в торренты, в шапке появляется кнопка управления

Watch Buttons — Плагин достаёт с кнопки "Смотреть" в полной карточке все кнопки просмотра в том порядке, в котором они находятся, и заменяет кнопки "Торренты" и "Трейлеры" на цветные
